# WalletConnect-in-Stacks-Blockchain---Smart-Tip-System-
WalletConnect in Stacks Blockchain - Smart Tip System 
This smart contract, developed in Clarity for the Stacks blockchain, is an advanced tipping and interaction system for dApps. It is specifically designed to integrate with AppKit (Reown/WalletConnect), providing a seamless user experience with on-chain loyalty logic.
